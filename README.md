# 📱 SaleMate App

SaleMate is a Flutter-based mobile application integrated with REST APIs for user authentication and data handling. The app demonstrates login functionality, API integration, token management, and clean UI design.

## 🚀 Features
- User Login using API
- Token-based authentication
- API integration (GET & POST)
- SharedPreferences for session management
- Clean and responsive UI
- Navigation between screens
- Error handling for API responses

## 🛠 Tech Stack
- Flutter (Frontend)
- Dart
- REST API
- SharedPreferences
- HTTP package

## ▶️ How to Run
1. Clone the repository  
2. Run `flutter pub get`  
3. Run `flutter run`

## 📦 Build APK
Run:
flutter build apk --release

APK location:
build/app/outputs/flutter-apk/app-release.apk
